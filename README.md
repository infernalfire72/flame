# flame

# An osu!bancho written in Go

# Setup
  You need a working MySQL setup

  1. `go get -t github.com/infernalfire72/flame`
  2. build
  3. run
  4. Edit the config file
  5. run

# Features
  - Bancho
  - Web:
    - Leaderboards, Screenshot Uploads
